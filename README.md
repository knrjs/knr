### 开发流程

1) 进入dev_mode，打开inspector，进入调试模式

```sh
  npm run dev     测试环境启动 自动唤醒浏览器
  npm run dev_no  测试环境启动 不唤醒浏览器
  npm run dev_pro 生产环境启动 自动唤醒浏览器
  npm run dev_pro_no 生产环境启动 不唤醒浏览器
```
